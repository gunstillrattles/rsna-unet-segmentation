# RSNA U-Net Medical Image Segmentation

This project is focused on medical image segmentation using a U-Net-based deep learning approach.

## Tech Stack
- Python
- PyTorch
- U-Net
- Jupyter Notebook

## Project Overview
The goal of this project is to perform medical image segmentation using deep learning techniques.  
The notebook includes data preparation, model training, evaluation and result analysis.

## Features
- Medical image preprocessing
- U-Net model training
- Segmentation evaluation
- Visualization of predictions

## Files
- `rsna_unet_segmentation.ipynb` — main notebook with full pipeline
